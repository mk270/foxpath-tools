Supported formats
=================

* X exists?
* X exists more than N times?
* X or X exists?
* only one of X or X exists?
* X is an T?
* X has more than N characters?
* X sum to N?
* X is in list L?

Where:
* L is the name of a list (e.g. a codelist)
* N is a number
* T is a type
* X is an XPath string
